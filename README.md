## Описание

Современный одностраничный сайт-портфолио с уникальным неоновым дизайном в фиолетово-розовой цветовой гамме. Сайт демонстрирует навыки и проекты, включает интерактивный 3D-персонаж и профессионально оформленные секции.

## Особенности

### Визуальные эффекты
-  **Анимированный Loading Screen** — кастомный экран загрузки с прогресс-баром и glitch-эффектами
-  **Неоновая цветовая схема** — градиенты от фиолетового (#c770f0) к розовому (#f07af0)
-  **Плавающие частицы** — фоновые анимации для создания атмосферы
-  **Glassmorphism эффекты** — прозрачные элементы с blur-эффектом

### Интерактивность
-  **3D Аватар на Three.js** — полностью анимированный персонаж с:
  - Моргание глаз
  - Отслеживание позиции мыши
  - Реакция на скролл
  - Анимация "дыхания"
-  **Карусели проектов** с Lightbox просмотром
-  **Адаптивное мобильное меню**

### Секции сайта
- **Hero** — приветственная секция с 3D-аватаром
- **About** — информация о разработчике
- **Skills** — навыки (Hard Skills с прогресс-барами, Soft Skills)
- **Work** — портфолио проектов
- **Contact** — контактная информация
